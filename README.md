# DDS Media Player Assignment

## Features
- JSON based layout rendering
- Dynamic zones based on x, y, width, height
- Image playback using Picasso
- Video playback using ExoPlayer
- Web content using WebView
- Media looping
- Media download & caching using DownloadManager

## Architecture
JSON → Parse → Dynamic Zones → ZoneMediaPlayer → MediaViewFactory

## Technologies Used
- Java
- Gson
- Picasso
- ExoPlayer
- WebView

## How to Run
1. Clone project
2. Open in Android Studio
3. Run on device
